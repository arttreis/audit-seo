Resumo Executivo

A Instrutherm é referência no mercado brasileiro de instrumentos de medição há mais de 35 anos, mas seu site não reflete essa autoridade. A loja virtual opera em Magento 2 com infraestrutura sólida (Cloudflare, HTTPS, HTTP/3), porém apresenta lacunas significativas que limitam sua visibilidade online e potencial de vendas.

O site está invisível para buscas com IA (ChatGPT, Perplexity, Google AI Overviews) e não possui dados estruturados (Schema), o que significa que produtos não aparecem com estrelas, preços ou disponibilidade nos resultados do Google. Além disso, mais de 40 categorias de produtos não possuem conteúdo descritivo, perdendo posições para concorrentes.

A boa notícia: todos os problemas são corrigíveis, e a base técnica existente (Cloudflare, URLs bem estruturadas, 700+ páginas indexadas) fornece uma fundação sólida para escalar resultados rapidamente.

Criar conteúdo descritivo nas 40+ categorias sem texto pode capturar uma fatia significativa desse tráfego.

Plano de Ação Priorizado

CRÍTICO — Corrigir imediatamente

1
Implementar Schema.org (JSON-LD)
Adicionar markup estruturado em todas as páginas: Organization (nome, logo, contato), Product (nome, preço, disponibilidade, imagem, marca, SKU) em cada produto, BreadcrumbList em páginas com breadcrumbs, e WebSite com SearchAction na homepage. Impacto: Rich results de produto no Google SERPs.
Esforço: Médio Impacto: ★★★★★
2
Corrigir hierarquia de headings (H1)
A homepage possui 5+ tags H1. Cada página deve ter exatamente 1 H1 com a proposta de valor principal. As categorias na homepage devem usar H2/H3.
Esforço: Baixo Impacto: ★★★☆☆
3
Adicionar meta descriptions únicas
Nenhuma meta description foi detectada na homepage ou páginas de categoria. Cada página precisa de uma descrição única e convincente (150-160 caracteres) para melhorar CTR nos resultados de busca.
Esforço: Baixo Impacto: ★★★★☆
4
Adicionar tags canonical
Nenhuma tag canonical detectada. Cada página precisa de uma canonical self-referencing para prevenir conteúdo duplicado, especialmente crítico com os filtros de URL do Magento.
Esforço: Baixo Impacto: ★★★★☆

ALTO — Corrigir em até 1 semana

5
Reduzir Crawl-delay de 15s para 1s
O Crawl-delay de 15 segundos limita severamente o crawl budget do Google. Reduzir para 1s ou remover completamente permite ao Googlebot indexar o site muito mais rápido.
Esforço: Baixo Impacto: ★★★☆☆
6
Melhorar TTFB (Time to First Byte)
TTFB de 1.37s está muito acima da meta de <800ms. Habilitar Full Page Cache do Magento, configurar Varnish ou Redis como backend de cache. Cache-Control está desabilitado (no-cache, no-store).
Esforço: Alto Impacto: ★★★★☆
7
Criar conteúdo descritivo nas categorias
Todas as páginas de categoria estão sem texto descritivo (thin content). Adicionar 150-300 palavras por categoria explicando os tipos de produto, aplicações e guia de compra.
Esforço: Médio Impacto: ★★★★☆
8
Atualizar datas do Sitemap
Todas as datas lastmod são de 2021-2022. Atualizar para refletir as modificações reais das páginas. Sitemaps com datas desatualizadas perdem credibilidade com crawlers.
Esforço: Baixo Impacto: ★★★☆☆
9
Adicionar header HSTS
Configurar Strict-Transport-Security: max-age=31536000; includeSubDomains para forçar conexões HTTPS e melhorar segurança.
Esforço: Baixo Impacto: ★★★☆☆

MÉDIO — Corrigir em até 1 mês

10
Converter imagens para WebP
Converter imagens de produtos para WebP com fallback JPEG para redução de 30-50% no tamanho dos arquivos.
Esforço: Médio Impacto: ★★★☆☆
11
Adicionar preload hints para recursos críticos
Preload de CSS crítico, fontes e imagens hero para melhorar LCP e FCP.
Esforço: Baixo Impacto: ★★★☆☆
12
Criar arquivo llms.txt
Adicionar /llms.txt com informações da empresa, categorias de produtos e páginas-chave para facilitar o acesso de crawlers de IA.
Esforço: Baixo Impacto: ★★★☆☆
13
Definir política de AI crawlers no robots.txt
Permitir explicitamente GPTBot e ClaudeBot no robots.txt para aumentar visibilidade em buscas com IA.
Esforço: Baixo Impacto: ★★★☆☆
14
Habilitar page caching
Remover no-cache/no-store para páginas estáticas e de categorias. Configurar cache headers adequados com Cloudflare.
Esforço: Médio Impacto: ★★★★☆
15
Ativar CSP (Content Security Policy)
CSP está em report-only. Migrar para modo enforced gradualmente para melhorar segurança.
Esforço: Médio Impacto: ★★☆☆☆
